# Customers
https://stripe.com/docs/api/customers

**List**
```
stripe customers list
```

**Create**
```
stripe customers create ...
```

**etrieve**
```
stripe customers retrieve <customer id>
```

# Payouts
https://stripe.com/docs/api/payouts

**List**
```
stripe payouts list
```

**Create**
```
stripe payouts create \
  --amount=1100 \
  --currency=usd
```
